How it works

Three steps. That's it.

No complicated setup, no IT degree required.

1

Install the agent

Download and run the RemoteMaster agent on the PC you want to access. Takes 30 seconds.

2

Set up your server

Point the agent to your home server (or just run it on the same PC). It handles the connection magic.

3

Open a browser

Go to your server URL from any device. Log in. You're now looking at your PC. Play, work, whatever.
